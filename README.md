# LevelCoach Training

LevelCoach Training is a sports-coaching marketplace and operations platform for athletes, coaches, and training organizations.

## Run Locally

```bash
npm install
npm run dev
```

Create `.env.local` with the Appwrite and payment settings needed for your environment. See `APPWRITE_SETUP.md` for Appwrite provisioning notes.

## Useful Commands

```bash
npm run build
npm run lint
npm run typecheck
```

## Product Surface

- Athletes and parents can discover coaches, book sessions, pay, message, and track progress.
- Coaches can manage profiles, availability, sessions, clients, earnings, and messages.
- Admins can manage users, coaches, pricing, bookings, credits, content, applications, conversations, and platform operations.
